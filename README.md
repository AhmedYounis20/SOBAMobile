### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ahmedyounis20/sobamobile.git
   ```
2. Install Yarn packages
   ```sh
   yarn
   ```
3. Run expo
   ```sh
     yarn start
   ```
